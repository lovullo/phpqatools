phpqatools
==========

Composer metapackage containing the following PHP QA and development tools:

* phpunit
* dbunit
* phpcs
* pdepend
* phpmd
* phpcpd
* phploc
* phpdox
* php-cs-fixer
* security-checker

Usage
==========

Simply require it in your project's `composer.json`

```json
    "require": {
        "lovullo/phpqatools": "@stable"
    }
```

Or

```sh
$ composer require lovullo/phpqatools
```
